How I Started Coding

I got interested in the concept of programming when I was still at middle school. There was an extracurricular course aptly named 'Programming' and I was very interested so i joined alongside with some of my friends who were also interested in coding. We started off with Web Development but we weren't introduced to professional code editors such as VSC. So we coded in the most traditional way with the lovely writing application that came with almost every laptop which is Notepad. Then after a while we continued to general coding concepts used on most backend programming languages like C++, Python, and JavaScript (Back then, I studied all that in C++). Till this day I still question how these coding concepts can be implemented in a workplace and actually create something useful. The only time I knew how was on an Arduino where I coded things to do on the board with C.

Fast forward to high school, we get more detailed on those coding concepts. This time we actually used a code editor (specificly a compiler since we still used C++) except we still didn't use the professional ones like VSC and instead used an online compiler called onlinegdb and we used that for the 3 years of programming and informatics class. However, in the middle of my highschool years, My parents suggested me to join this well known coding course. I joined and we actually studied both WebDev and general coding concepts in an even more detailed version than the ones i studied so far at school. I also got to use VSC in the process and thats how I finally managed to get my hands on an actual professional code editor. Since then, I have never stopped using VSC as now that I have graduated to college and entered my most desired major, which is Computer Science. At this point, I can't work without VSC.

Getting back to the topic of my progress being a programmer, now that I'm in college studying Computer Science, I finally got to do more coding than having to suffer from useless knowledge that I probably wont use for the rest of my life. At this point, all the coding concepts are reviewed but now it's gotten really deep into the nooks and cranies of coding. But regardless, I still get to learn new things as I haven't even scraped the surface of what I could do in the programming world. It's a lot of fun but it's the kind of fun where you should know what you can do as it is very blurry from the start
